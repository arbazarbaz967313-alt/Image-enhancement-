What Snap Enhance Can Do

Four powerful AI enhancements, all free and running on real open-source models.

⬆️
AI Upscaling
Real-ESRGAN super-resolution delivers 2× or 4× enlargements. Lanczos fallback ensures it always works.
Real-ESRGAN
👤
Face Restoration
GFPGAN recovers facial details from blurry or old photos. OpenCV CLAHE fallback always available.
GFPGAN + OpenCV
✨
Smart Denoising
Non-Local Means removes grain and JPEG artifacts while keeping edges sharp.
NLM Color
🔬
Precision Sharpening
Unsharp Mask applied last in the pipeline for maximum detail recovery.
Unsharp Mask
Processing Pipeline
📥 Input→ ⬆️ Upscale→ ✨ Denoise→ 👤 Face→ 🔬 Sharpen→ 📤 Output
Each step is isolated — if one fails, the rest continue normally.

📡 REST API Reference

Send images via multipart POST, receive enhanced PNG/JPEG in response.

🔗 https://bg-of-image-enhancement.onrender.com
Health Check
GET
GET /health

# Response
{ "status": "ok", "capabilities": { ... } }
Enhance Image
POST
POST /enhance
  ?upscale=true&scale=2
  &face_restore=true
  &denoise=true&sharpen=true

# Body: multipart/form-data
file: <image file>

# Response: image/png
cURL Example
curl -X POST \
  "https://bg-of-image-enhancement
  .onrender.com/enhance
  ?upscale=true&scale=2" \
  -F "file=@photo.jpg" \
  -o enhanced.png
Parameters
ParamTypeDefaultInfo
upscalebooltrueEnable upscaling
face_restorebooltrueRestore faces
denoisebooltrueRemove noise
sharpenbooltrueSharpen image
scaleint22 or 4
output_formatstrPNGPNG or JPEG
