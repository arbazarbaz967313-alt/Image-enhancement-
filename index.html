Enhance Image
POST 
# cURL — all features, 2x scale
curl -X POST \
  "http://localhost:8000/enhance?\
upscale=true&scale=2&\
face_restore=true&\
denoise=true&sharpen=true" \
  -F "file=@photo.jpg" \
  -o enhanced.png
JavaScript Fetch
POST 
const fd = new FormData();
fd.append('file', imageFile);

const res = await fetch(
  `http://localhost:8000/enhance
    ?upscale=true&scale=2
    &face_restore=true
    &denoise=true&sharpen=true`,
  { method: 'POST', body: fd }
);
const blob = await res.blob();
const url = URL.createObjectURL(blob);
Python Requests
POST 
import requests

with open("photo.jpg", "rb") as f:
  res = requests.post(
    "http://localhost:8000/enhance",
    params={
      "upscale": True,
      "scale": 2,
      "face_restore": True,
      "denoise": True,
    },
    files={"file": f},
  )

with open("out.png","wb") as f:
  f.write(res.content)
